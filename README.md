## 🛠️ 技术栈

### 必学核心
- Python 异步编程（async/await）
- LangChain / LangGraph
- LLM API（OpenAI、Anthropic、本地模型）
- Vector Database（Chroma、Pinecone）
- Agent 设计模式（ReAct、Plan-and-Solve）

### 加分项
- AutoGen / CrewAI
- FastAPI（部署 Agent 服务）
- Docker 容器化
- LangSmith（监控与日志）
